# Glimpaxe Momento — Hero Landing Page

A focused, single-section hero landing page built for **Glimpaxe Momento**, a Lagos-based business specializing in premium, customized event souvenirs and corporate gifts.

🔗 [Live site](https://afodunrinbikunmi-data.github.io/glimpaxe-website): afodunrinbikunmi-data.github.io/glimpaxe-website

# About the project

This was built as client/freelance work — a streamlined, single-screen hero page designed to make a strong first impression and drive quick action, rather than a full multi-section site. It pairs a bold headline and value proposition with a branded product showcase image.

The page was developed from a UI design brief and translated into responsive, semantic HTML and CSS, matching the client's brand colors, typography, and layout.

# Sections


- **Header** — navigation bar with brand logo and "Get Started" CTA
- **Hero** — eyebrow label, headline, supporting copy, dual call-to-action buttons (View Collection / Chat on WhatsApp), and a three-point feature row (Premium Quality, Fast Delivery, Fully Customized)
- **Product showcase** — branded promotional graphic of the gift package collection
- **Scroll indicator** — animated cue for continued exploration


# Tech stack


- HTML5
- CSS3 (Grid & Flexbox layout, no framework)
- Vanilla JavaScript (mobile navigation toggle)
- Google Fonts (Poppins, Inter)


No build tools, frameworks, or dependencies — a single self-contained, fast-loading static page.

# Features


- Fully responsive across desktop, tablet, and mobile
- Direct WhatsApp integration on call-to-action buttons
- Decorative background elements (dot pattern, soft gradient circle) for visual depth
- Optimized semantic HTML for accessibility and SEO
- Lightweight — no external JS libraries
